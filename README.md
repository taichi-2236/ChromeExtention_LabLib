# GoogleChrome拡張機能「LabLib」
インターネット上で閲覧しているPDFを収納，管理できる拡張機能

### 使用方法
1. 保存したい論文のリンクをChrome上で開き，LabLibのアイコンをクリックする
2. 「保存」ボタンを押し，タイトルとタグを入力して保存を押す

### 機能
- 保存した論文の表示
- 詳細ページから保存内容の編集，リンクのコピー，ダウンロード，削除
- 各論文へのタグ付
- 論文のタイトル，タグでの検索
- タグをクリックすることで検索欄にタグを自動入力
- 複数選択による一括削除，ダウンロード，参考文献リスト作成
- 同一Googleアカウント間での保存データ共有

### 注意
データの保存に`chrome.storage.sync`を利用しており，100KBの制限があるため保存できる量に限りがある．
サイドパネル上部の使用容量は，保存した論文が占めている容量を表している．
保存容量をカーソルオーバーすることで容量の詳細を確認することができる．
